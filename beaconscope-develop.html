Based on Open-Source Components

This app is based on the open-source Android Beacon Library, which makes it easy to build beacon apps for Android using Kotlin and Java code. This is the most widely used library for building beacon apps on Android. Learn more about this at the project website.

Build Your Own Beacon App

You can use the Android Beacon Library's free reference application, with source code in Kotlin, as a starting point to build your own Android beacon scanning app. It's published on Github, so you can download it for free and get started with it today.
